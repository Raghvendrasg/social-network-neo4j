# Social Network with FastAPI & Neo4j Aura

A simple **social network backend** built with **FastAPI** and **Neo4j Aura** as the graph database. This project demonstrates creating users, posts, and connections using **graph-based relationships**, providing a fast and scalable way to model social networks.

---

## 🚀 Features

- User management: create, update, and delete users
- Post management: add, edit, and retrieve posts
- Social connections: follow/friend relationships between users
- Graph database backend powered by **Neo4j Aura**
- Async API support with **FastAPI**
- Swagger docs auto-generated at `/docs`

---

## 🛠 Tech Stack

- **Backend:** [FastAPI](https://fastapi.tiangolo.com/)
- **Database:** [Neo4j Aura](https://neo4j.com/cloud/aura/)
- **Python:** 3.10+
- **Virtual Environment:** `venv`
- **HTTP Server:** Uvicorn
